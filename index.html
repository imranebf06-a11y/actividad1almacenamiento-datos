Tipos principales

Almacenamiento Primario

Memoria RAM, Caché y registros de CPU. Muy rápido, pero volátil.

Almacenamiento Secundario

Discos duros, SSD, cintas. Más lento pero persistente.

Bases de Datos

Sistemas organizados para almacenar y consultar datos estructurados.

Almacenamiento en la Nube

Servicios externos que guardan datos en Internet con escalabilidad.

Glosario

  • RAM: Memoria de acceso aleatorio, rápida y volátil.
  • SSD: Unidad de estado sólido, almacenamiento persistente rápido.
  • Base de datos: Sistema para organizar y consultar datos.
  • Cloud: Infraestructura remota para guardar datos en Internet.
